# Specifications HDMI — Sony HT-S40R

**Date :** Non specifie  
**Contexte :** Barre de son home cinema  
**Type :** Systeme audio HDMI ARC

## Vue d'ensemble

Le HT-S40R est une barre de son home cinema de gamme budget, equipee d'une sortie HDMI ARC pour la connexion au televiseur. Elle supporte les formats audio standards mais ne dispose pas d'entrees HDMI.

## Specifications techniques

### Configuration HDMI

| Specification | Detail |
|---|---|
| **Type de connexion** | HDMI ARC (Audio Return Channel) |
| **Nombre d'entrees/sorties** | 0 entrees HDMI / 1 sortie HDMI (ARC) |
| **HDMI CEC** | Oui (controle par CEC) |
| **Version HDMI** | HDMI 2.0 |

### Formats audio supportes

| Format | Support |
|---|---|
| **Dolby Digital (5.1)** | ✓ Oui |
| **LPCM 2 canaux** | ✓ Oui |
| **Dolby Dual Mono** | ✓ Oui |
| **Dolby TrueHD** | ✗ Non |
| **Dolby Atmos** | ✗ Non |
| **DTS-ES / DTS-HD** | ✗ Non |

## Limitations importantes

### ⚠️ Absence d'entree HDMI

Le HT-S40R **n'a pas d'entree HDMI**. Il dispose uniquement d'une **sortie HDMI ARC**.

Cela signifie :
- La barre se connecte au **televiseur via HDMI ARC** (pas l'inverse)
- La TV doit envoyer l'audio a la barre de son par ce meme cable
- Les formats supportes concernent ce que le TV peut envoyer a la barre

## Notes techniques

Les specifications officielles Sony ne mentionnent pas le support de :
- **Dolby Digital Plus (DD+)**
- **Formats haute resolution** (DTS:X, Atmos, TrueHD)

Ce positionnement est coherent avec la gamme budget du produit.
