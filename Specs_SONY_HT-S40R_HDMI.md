# Spécifications HDMI — Sony HT-S40R

## Configuration HDMI

| Spécification | Détail |
|---|---|
| **Type de connexion** | HDMI ARC (Audio Return Channel) |
| **Nombre d'entrées/sorties** | 0 entrées HDMI / 1 sortie HDMI (ARC) |
| **HDMI CEC** | Oui (contrôle par CEC) |
| **Version HDMI** | HDMI 2.0 |

## Formats audio supportés

| Format | Support |
|---|---|
| **Dolby Digital (5.1)** | ✓ Oui |
| **LPCM 2 canaux** | ✓ Oui |
| **Dolby Dual Mono** | ✓ Oui |
| **Dolby TrueHD** | ✗ Non |
| **Dolby Atmos** | ✗ Non |
| **DTS-ES / DTS-HD** | ✗ Non |

## ⚠️ Limitation importante

Le HT-S40R **n'a pas d'entrée HDMI**. Il dispose uniquement d'une **sortie HDMI ARC**.

Cela signifie :
- La barre se connecte au **téléviseur via HDMI ARC** (pas l'inverse)
- La TV doit envoyer l'audio à la barre de son par ce même câble
- Les formats supportés concernent ce que le TV peut envoyer à la barre

## Note technique

Les spécifications officielles Sony ne mentionnent pas le support de :
- **Dolby Digital Plus (DD+)**
- **Formats haute résolution** (DTS:X, Atmos, TrueHD)

Ce positionnement est cohérent avec la gamme budget du produit.
